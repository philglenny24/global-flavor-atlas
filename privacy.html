Privacy Policy

Last updated: April 17, 2026

1. Information We Collect

The Global Flavor Atlas is a frontend-only project. We do not collect, store, or share any personal identification information (PII) from our users.

2. Third-Party Services

Our website uses a public API to provide culinary information. When you use the "Explore" page, your browser makes a request to a third-party service. Please note that these services may collect your IP address as part of the standard web request process.

3. Deployment

This site is hosted via GitHub Pages and deployed using GitHub Actions. GitHub may collect certain data for security and analytical purposes in accordance with their own privacy standards.

4. Contact

For questions regarding this project, please contact the developer via the GitHub repository.

5. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.
